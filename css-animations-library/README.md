# CSS Animations Library

37 curated CSS animation examples organized into 8 categories, sourced from [Prismic's CSS Animation Examples](https://prismic.io/blog/css-animation-examples).

## Categories

| Category | Count | Description |
|----------|-------|-------------|
| Text | 4 | Flip, gradient, reveal, glitch |
| Hover | 5 | Zoom, underline, 3D tilt, magnetic, ripple |
| Button | 5 | Neon glow, border, fill slide, shake, morph |
| Click | 5 | Heart burst, checkbox, toggle, SVG draw, confetti |
| Loading | 5 | Dots, progress bar, pulse, skeleton, spinner |
| Menu | 5 | Hamburger, slide-in nav, dropdown, stagger, morph |
| Card | 5 | Flip 3D, slide reveal, stacked, lift, glass |
| Other | 3 | Scroll bar, particle burst |

## Usage

Open `index.html` in a browser to browse all animations as live CodePen embeds.

Each card shows the animation running live and links directly to its CodePen source so you can copy the HTML/CSS/JS into your project.
